## 📖 Project Overview

This repository documents my journey to master Computer Science and Mathematics from foundational concepts to advanced applications in biomechanics, sports science, and wearable technology. 

### 🎯 Mission

Develop the technical expertise required to work in R&D roles at leading sports technology companies (Whoop, Oura, Garmin, Brooks, Nike) or as a Sport Scientist/Data Analyst in high-performance organizations, while simultaneously preparing for PhD programs in Kinesiology or Biomedical Engineering.

### 🚀 What Makes This Unique

- **Complete Reconstruction:** Starting from basic arithmetic, building to advanced machine learning and computer vision
- **Applied Focus:** Every concept connected to human performance, biomechanics, and sports analytics
- **Portfolio-Driven:** All learning manifests in tangible projects demonstrating practical skills
- **Public Accountability:** Transparent progress tracking, code quality, and continuous improvement
- **Interdisciplinary:** Bridging exercise science, data science, software engineering, and biomechanics

## 🎓 Learning Philosophy

This project follows **Scott Young's Ultralearning** framework:

1. **Meta-learning:** Curriculum based on top CS programs (MIT, Stanford, CMU)
2. **Focus:** 11 dedicated hours weekly, distraction-free learning
3. **Directness:** Project-based learning with real biomechanics applications
4. **Drill:** Targeted practice on weak areas
5. **Retrieval:** Active recall through problem-solving
6. **Feedback:** Self-assessment and continuous improvement
7. **Retention:** Spaced repetition, progressive complexity
8. **Intuition:** Deep understanding before moving forward
9. **Experimentation:** Apply concepts to real research data
